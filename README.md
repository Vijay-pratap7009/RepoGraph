RepoGraph is a visualization and management tool for Git repositories.  
It helps developers **understand commit history, branching, and collaboration patterns** through intuitive graphical representations.

---

## 🚀 Features
- Visualize commit history as interactive graphs
- Track branches and merges with clarity
- Analyze contributor activity
- Lightweight CLI integration
- Works seamlessly with existing Git repos
